# I Am Zee Shop - GeneratePress Child Theme

This child theme customizes the GeneratePress theme for the "I Am Zee Shop" brand. It provides specific styling and functionality for our e-commerce store, which is built on WordPress and WooCommerce, ensuring a unique and consistent brand experience.
